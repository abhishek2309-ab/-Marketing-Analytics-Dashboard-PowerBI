## 🎯 Project Overview

An interactive Power BI dashboard designed to analyze marketing campaign performance, customer acquisition costs, ROI, lead generation, conversion rates, and channel effectiveness.

|----------|----------|

| 📈 ROI | 6.60K |
| 🔄 Conversion Rate | 24.39% |
